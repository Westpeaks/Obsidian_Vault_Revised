
## General System Information 

### Who Am I?

- This is the current user you are logged in as. 
```bash
whoami
```
### What Machine is This?

- This will tell you the current machine you are logged into.
```bash
hostname
```
### Where Am I?

- This will tell you the current directory you are logged into. Command stands for print working directory
  ```bash
  pwd
  ```
### What is in the Contents of a Directory?

- This will list the files in the current directory.
  ```bash
  ls
  ```
- To list the files of another directory, you would add the file path.
  ```
  ls /etc
  ```
