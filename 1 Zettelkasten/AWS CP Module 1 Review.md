## Client Server Model

In the client-server model, the client sends requests to the server, which processes the requests and sends back responses. Cloud computing provides scalable server resources that can be accessed over the internet.

## Cloud Computing Model
  
Cloud computing is a model for delivering IT resources over the internet. Businesses can rent services on a pay-as-you-go basis, which can help reduce fixed costs and allow for more flexible budgeting.

## Shared Responsibility Model
  
AWS is responsible for securing the physical infrastructure, companies can focus on securing their data and applications within the cloud.

Customers manage security **in** the cloud. This includes:
- Managing OS patches
- Encrypting client-side data
## Hybrid Model Example

A government agency wants to maintain complete control over its IT infrastructure, but plans to use AWS Cloud services for specific applications.

## Benefits of AWS Global Infrastructure

- Fault tolerance
- High availability

## AWS Structure for High Availability and Fault Tolerance

A Region is a geographical location that contains three or more Availability Zones. An Availability Zone is a distinct location within a Region that contains one or more discrete data centers.
