EC2s are the AWS version of a virtual machine. These instances can be used and specialized for any number of specific tasks. 

- **General Purpose** - ideal for diverse and standard workloads
- **Compute Optimized** - ideal for compute heavy tasks (**Ex.** gaming, machine learning)
- **Memory Optimized** - ideal for memory intensive tasks (**Ex.** data analytics, databases)
- **Accelerated Computing** - specifically engages hardware accelerators (**Ex.** floating-point calculations, graphics processing)
- **Storage Optimized** -  You guessed it, storage. Also, large databases.
