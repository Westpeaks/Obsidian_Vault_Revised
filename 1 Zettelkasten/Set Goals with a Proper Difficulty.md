Although failure can be a driving force, goal setting should reflect the following. If you want to meet a goal everyday:

	Pick a goal which is:
		- Stimulating and challenging enough.
		- Contains a healthy amount of anxiety and difficulty.
		- It should be daunting, but not overly challenging. 
		- It should be acheivable.
