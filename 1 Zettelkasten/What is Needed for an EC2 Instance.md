- EC2 instances are launched using Amazon Machine Instances (**AMI**). These are pre-built virtual machines that require the configuration of several parameters.
- The key components to configure and consider when configuring and EC2 AMI:
	- Name
	- The AMI to be used
	- OS
	- Instance type (how much computing power the AMI will used)
	- Key pair (how we will log into the instance)
	- Network settings
	- Storage
	- Advanced details (this is where you can customize the functionality of the instance **Ex.** web server)

- You can also create your own custom AMIs.
