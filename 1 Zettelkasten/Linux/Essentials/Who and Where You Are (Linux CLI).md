# whoami

This command reveals who the current user is and can be valuable for server administration.

```shell
wesleypeak@wesleys-mbp  ~  whoami
wesleypeak
```

# pwd

This will reveal the current directory path that the user is in in the terminal. 

```shell
wesleypeak@wesleys-mbp  ~  pwd
/Users/wesleypeak
```
