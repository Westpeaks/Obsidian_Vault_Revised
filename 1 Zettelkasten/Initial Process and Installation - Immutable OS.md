## Creating Bootable Drive for Installation 

1. Navigate to the [Fedora Cosmic Atomic Desktop](https://fedoraproject.org/atomic-desktops/cosmic/download/) page.
2. Download the current iso file.
3. Use the latest version of [belenaEtcher](https://etcher.balena.io) to burn the iso on to a bootable drive.
	- Following the steps within the app is straight forward. 
4. Boot up the computer that will be used and hold down the keyboard key that will initiate the BIOS or UEFI Boot Settings. 
	- On Thinkpads this is usually the F12.
5. Select the USB drive to launch the Fedora installation configuration process. 
6. Follow the installation settings and go through the install until you reach the initial Cosmic screen and log in.
