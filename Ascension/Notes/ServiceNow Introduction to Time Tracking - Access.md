## Access

### From SNow

- Click **All** and use search in the **Filter Navigator**.
- Type "time sheet".
- Select **Time Sheet Portal**.
- **Warning** do not use **My Current Time Sheet**.
- You can save the time sheet portal as a favorite for convenience. 

### Access Errors

#### How to resolve:

- Contact your manager. 
- Alternatively, you can create a ticket using an SNow catalog item to request membership to a resource group. The most common cause of an access error is that you are not a member of a resource group.
- Your manager or supervisor can help determine the appropriate resource group.
