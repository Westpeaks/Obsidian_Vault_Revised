## Resolution and Recovery

### Resolve an incident

To resolve an incident, use the following steps:

1. Change the Incident State
2. Complete the Resolution Information
3. Create a Knowledge Article (KB) (if warranted)
4. Choose a Resolution Code
5. Describe the Resolution
6. Confirm the Resolution

### Resolve the Parent Incident

If incidents involve a parent child relationship, it is important to close the parent incident ticket. 

This will close all associated child incidents. 

### Reopen an Incident

If the caller is unsatisfied with a resolution, an incident can be reopened. 

#### To reopen an incident:

- respond to the resolution email
- Use MyTechnology
- Contact the Service Desk
- Manually change the state back to "In Progress"
