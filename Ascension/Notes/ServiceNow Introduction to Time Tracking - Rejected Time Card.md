## Rejected Time card

### Why a time card might be rejected

- The Project Manager disagrees the Project Time Category you selected. 
- The time was added to the wrong project or wrong week. 
- There is a mistake in the hours entered.

### How to fix it

1. Go back to the Time Sheet Portal.
2. Find the rejected time card.
3. Correct the issue. 
4. Resubmit the corrected time sheet.
