## Getting Started with MyTechnology

### What is MyTechnology?

It is a portal that is the primary source for associates to get help. You can:

- Report a technology issue
- order hardware
- request software/applications
- search KB articles
- track your active tickets

### Navigation

To access MyTechnology, go to Good Day Ascension (web page).

You can also access it from ServiceNow (search for MyTechnology).

In MyTechnology you can:

- Search for catalog items and KB's.
- Report an issue. 
- Reset passwords.
- Check your active items. 
- Favorite KB's and request items.
