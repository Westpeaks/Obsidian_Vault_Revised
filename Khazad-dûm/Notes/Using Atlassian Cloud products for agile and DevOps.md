# Using Atlassian Cloud products for agile and DevOps
<H2>Atlas</h2>
A directory oriented towards teamwork on a centralized platform. 
- Create a hub
- Define the status and goals
- See who is working on what
- Easily find and share information
- Stay up-to-date on the latest changes 
Atlas is most used by product owners, project managers, development teams, and leadership teams. 

<h2>Bitbucket</h2>
An online version control repository. 
- Store and manage Git repositories
- Track changes
- Review and merge
- Automate build and deployment
Bitbucket is used by software developers. 

<h2>Compass</h2>
A guide to navigating your distributed software architecture. It's a platform for cataloguing, monitoring, and handling all of your software components and their dependencies as well as the teams behind them.
Compass is used by software developers. 

<h2>Confluence</h2>
Confluence is a knowledge repository platform. Confluence allows for teams to create and share content. It is for creating and sharing documentation across the organization.
Confluence is used by all teams.
