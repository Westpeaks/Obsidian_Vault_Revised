# Interview Red Canary Info


![[RCMM1.svg]]

### Red Canary offers several software solutions as part of their cyber security services:

- **SaaS-based MDR Platform:** A comprehensive Managed Detection and Response platform that integrates software technology with expert cyber security services.

- **Proprietary EDR Sensor:** A lightweight agent designed specifically for Linux environments, optimized to use minimal resources.

- **Built-in SOAR Platform:** Security Orchestration, Automation, and Response platform with customizable playbooks for faster threat response.

- **Atomic Red Team:** An open-source library of tests for simulating adversarial activity and validating defenses.

- **Red Canary Copilot:** Included in the Complete package, though specific details are not provided.

- **Canary Exporter:** A data export tool available in the Complete plan, allowing customers to export data from Red Canary to various destinations in standard formats.

- **File Integrity Monitoring:** Included in the Complete package.
